> **5 minutes to understand your AI.**

BotMark is the comprehensive benchmarking platform for AI agents, evaluating across **5 dimensions**:

- 🧠 **IQ** - Cognitive Intelligence (reasoning, coding, knowledge)
- ❤️ **EQ** - Emotional Intelligence (empathy, personality)
- 🛠️ **TQ** - Tool Intelligence (API usage, execution)
- 🛡️ **AQ** - Adversarial Resilience (safety, reliability)
- 🌱 **SQ** - Self Evolution (learning, reflection)

## 🚀 Quick Start

```bash
# Install BotMark CLI
npm install -g botmark-cli

# Run benchmark
botmark benchmark --agent your-agent
```

Or use our **web platform**: [botmark.cc](https://botmark.cc)

## 📊 What You Get

- **IQ·EQ·TQ·AQ·SQ** Scoring (0-1000)
- **MBTI Personality Analysis** for your AI
- **Detailed Optimization Reports**
- **Global Leaderboard** - See how you rank
- **Improvement Suggestions** based on your use case
